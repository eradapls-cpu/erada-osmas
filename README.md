# أوسماس OSMAS V2

نسخة مطورة جاهزة للرفع على Vercel كملف HTML واحد.

## الجديد
- قيمة حجز لكل طبيب.
- رفع صورة إشعار بنكك إلزامي.
- لا يتم إصدار التذكرة بدون الإشعار.
- تحديد عدد التذاكر اليومية لكل طبيب.
- لوحة إعدادات لتعديل الأطباء والأسعار والحد اليومي.
- عرض إشعار بنكك من لوحة الاستقبال.
- تصدير الحجوزات CSV يفتح في Excel.

## الرفع على Vercel
ارفع الملفات إلى GitHub ثم اختر المشروع من Vercel.
