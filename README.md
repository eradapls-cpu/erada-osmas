# أوسماس OSMAS

نسخة جاهزة للرفع على Vercel بدون React وبدون npm.

## طريقة الرفع
1. ارفع ملف index.html إلى GitHub داخل Repository جديد.
2. افتح Vercel واختر Add New Project.
3. اختر المستودع واضغط Deploy.
4. لا تحتاج Build Command ولا Output Directory.

هذه نسخة Static HTML تحفظ البيانات داخل المتصفح LocalStorage.
